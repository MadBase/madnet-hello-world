## Getting Started

This tutorial has two paths to utilize in learning how MadNet works. 

The general path is the same. You'll generate a wallet, fund it, and explore sending transactions and storing data on MadNet's testnet.

### [UI Guided Tutorial](/docs/UI%20-%20Tutorial/create-a-wallet)

This tutorial path shows what is possible while abstracting the underlying MadNetJS layer.

You won't look at code here, but you'll be able to interact with MadNet!

With the UI Guided tutorial you will be able to:

- Generate a wallet for use with MadNet
- Connect to testnet
- Fund your new wallet from the testnet faucet
- Learn About TXs on MadNet
- Send value TXs to other wallets
- Create and view data stores*

<sub>*Data stores are an integral part of the MadNet chain and how data is stored on-chain.</sub>
